**快捷操作**

-	复制
	-	`command + c` and `command + v`
	-	`command + d`
	-	`option + drag`
-	撤销
	-	`command + z`ß
-	调整角度

	-	`command + leftkey`
	-	`shift + leftkey`
-	成组

	-	`option + command + g`
	-	`option + command + shift + g`
-	锁定

	-	`command + l`
	-	`option + command + l`
-	全选/多选/剔除

	-	`command + a`
	-	`shift + left`
	-	`command + left`

**导航器**

-	复制和新建幻灯片

	-	`command + n`
	-	`command + d`
	-	`option + drag`
-	跳过幻灯片

	设置为跳过的幻灯片不播放，到处时有选项。应用场景是对某个页面有多种方案。
-	层级关系（基于树状结构）

**自定义快捷键**

`System Preference` --> `keyboard` --> `Shortcuts` --> `App Shortcuts` --> `+` --> 在`applications`中选择`keynote`依照提示设置即可。

**格式中高级属性**

-	样式
-	媒体占位符，修改对象时保存动画属性
-	成组构建的动画效果保存，双击成组对象后，直接新建构建。
